## 📱 Run on Termux

<details>  
<summary>📌 Termux Installation (CLICK TO EXPAND)</summary>  

```bash
# Update & install dependencies
pkg update && pkg upgrade
pkg install git nodejs -y

# Clone repo
git clone https://github.com/Shafiullah90/Shafi-king-bot.git

# Go inside folder
cd Shafi-king-bot

# Install packages
npm install

# Start the bot
node index.js
```
